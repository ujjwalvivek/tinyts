# TinyTS v0.2.2 - Release Notes

- Exported `Voice` class, `Renderer`, `SpriteOptions`, `TextOptions`, `InputManager`, `CanvasManager`, `CanvasOptions`, `CanvasResizeListener`, `TweenManager`, `ParticleShape`, `ColorStop`, `Point2D`, and `QueryConfig` for generated docs.
- Added `deploy` script to build game, stage docs, and serve locally.
- Added a tech demo game that showcases the engine's capabilities.
